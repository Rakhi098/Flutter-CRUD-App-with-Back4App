        <h1>Task Manager App – A Flutter-Based CRUD Application Using Back4App (BaaS)</h1>
        
        <h2>📌 Project Description</h2>
        <p>The <strong>Task Manager App</strong> is a <strong>Flutter-based CRUD application</strong> that utilizes <strong>Back4App</strong> as a Backend-as-a-Service (BaaS). It enables users to <strong>authenticate, manage tasks, and synchronize data</strong> seamlessly with a cloud backend, eliminating the need for a custom server.</p>
        
        <h2>🚀 Why Use Back4App (BaaS)?</h2>
        <ul>
            <li>✅ No need to manually set up or manage a backend.</li>
            <li>✅ Ready-to-use <strong>authentication and database services</strong>.</li>
            <li>✅ <strong>Real-time data storage and retrieval</strong> for seamless user experience.</li>
            <li>✅ Quick deployment and easy scalability.</li>
        </ul>
        
        <h2>✨ Features</h2>
        <ul>
            <li>✔ <strong>User Authentication</strong> – Register and login using a student email ID.</li>
            <li>✔ <strong>Task Management (CRUD)</strong> – Create, Read, Update, and Delete tasks.</li>
            <li>✔ <strong>Cloud Database</strong> – Data is securely stored in <strong>Back4App</strong>.</li>
            <li>✔ <strong>Real-Time Syncing</strong> – Changes reflect instantly in the database.</li>
            <li>✔ <strong>Secure Logout</strong> – Users can safely log out.</li>
        </ul>
        
        <h2>🛠️ Technology Stack</h2>
        <ul>
            <li><strong>Frontend:</strong> Flutter (Dart)</li>
            <li><strong>Backend:</strong> Back4App (Parse Server)</li>
            <li><strong>Database:</strong> Back4App Cloud Database</li>
            <li><strong>Version Control:</strong> GitHub</li>
            <li><strong>Development Platform:</strong> Local device</li>
        </ul>
        
        <h2>📜 App Workflow</h2>
        <h3>1️⃣ Authentication</h3>
        <ul>
            <li>Users register and log in using their <strong>student email ID</strong>.</li>
            <li>Credentials are securely managed through <strong>Back4App's authentication system</strong>.</li>
        </ul>
        
        <h3>2️⃣ Task Management</h3>
        <ul>
            <li>Users can <strong>create tasks</strong> with <strong>title and description</strong>.</li>
            <li>Tasks are stored in <strong>Back4App’s cloud database</strong>.</li>
            <li>Users can <strong>edit or delete tasks</strong>, with <strong>real-time updates</strong>.</li>
        </ul>
        
        <h3>3️⃣ Logout</h3>
        <ul>
            <li>Users can <strong>securely log out</strong>, ensuring session termination.</li>
        </ul>
